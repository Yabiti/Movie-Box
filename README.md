# 🎬 Movie Box

A modern, high-performance movie discovery application built with **Next.js** and **Tailwind CSS**. This app allows users to explore trending movies and search for their favorites in real-time using the **TMDB API**.

## 🚀 Features

- **Real-time Search:** Instantly find movies as you type.
- **Dynamic Hero Section:** A beautiful, responsive header showcasing trending titles.
- **Performance Optimized:** Implements Next.js `Image` optimization and LCP (Largest Contentful Paint) best practices.
- **Responsive Design:** Fully functional on mobile, tablet, and desktop.
- **Modern UI:** Built with Tailwind CSS and Lucide React icons for a sleek look.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com)
- **Icons:** [Lucide React](https://lucide.dev)
- **API:** [The Movie Database (TMDB)](https://themoviedb.org)
- **Language:** TypeScript
